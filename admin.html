¿Cómo cargar un listado nuevo?
El procesamiento con IA se hace en Claude, no acá. Esto evita errores de conexión y funciona siempre.
1
Copiá el texto del listado que te mandó el proveedor (WhatsApp, mail, lo que sea)
2
Abrí Claude y pedile: "Procesá este listado y devolveme el JSON para importar en mi panel" — pegá el texto abajo del mensaje
3
Claude te va a devolver el JSON comparado con tu catálogo actual + el JSON listo para importar
4
Copiá ese JSON y usá el botón "Importar JSON" de abajo para cargarlo acá
5
Revisá los cambios, ajustá lo que necesites y descargá el index.html actualizado
